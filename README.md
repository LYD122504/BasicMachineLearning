> ## 项目简介
>
> 本仓库记录了我系统学习机器学习理论的过程。主要参考Standford的 **CS229** 课程和李航老师的 **《统计学习方法》**。
>
> 不同于直接调用 `sklearn` 库，本项目的核心目标是**深入理解算法原理**。每个算法都包含：
> 1.  **数学推导**：详细的公式推导与理论笔记 (`Note.md`)。
> 2.  **从零实现**：使用基础代码编写核心算法逻辑，统一接口风格
> 3.  **实验验证**：在经典数据集上的运行结果与可视化。
